An automated workflow for professional photographers to extract image metadata into searchable taxonomies and apply cinematic watermarks.

== Description ==

The Automated EXIF-to-Taxonomy Cinematic Watermarker streamlines professional media management by automating the extraction of camera settings (Make, Lens, Focal Length, ISO) and converting them into indexable WordPress taxonomies. 

Additionally, the plugin uses the PHP GD library to programmatically generate a minimalist, cinematic watermark overlay on the 'large' image size, ensuring your original high-resolution files remain untouched while your web-ready previews are branded and informative.

== Installation ==

1. Upload the automated-exif-watermarker folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. (Optional) Place your preferred TTF font in /assets/fonts/cinematic-font.ttf to customize the watermark look. 
4. Upload an image to the Media Library to see the automation in action.

== Frequently Asked Questions ==

= Does this modify my original high-resolution uploads? =
No. The plugin specifically targets the 'large' image size generated by WordPress. Your original 'full' resolution file is left completely untouched and un-watermarked.

= What happens if an image has no EXIF data? =
The plugin handles this gracefully. If no EXIF data is detected, the taxonomy assignment and watermarking steps are simply skipped.

= Can I use my own font for the watermark? =
Yes. Simply replace the file at assets/fonts/cinematic-font.ttf with any TrueType Font (.ttf) of your choice. Ensure the filename remains exactly the same.
